# Amazon Report Analyzer 开发记录

## 开发内容

### 1. 主要功能
- 自动检测报告目录和文件类型
- 智能识别报告周期（年/月/季度）
- 生成可视化 HTML 报告
- 支持筛选和排序 SKU 明细
- 核心数据：销售额、订单、客单价、转化率、广告数据
- 退货分析

### 2. 支持的报告类型
| 类型 | 文件名关键词 |
|------|-------------|
| Business Report | BusinessReport |
| Transaction Report | Transaction |
| Returns Report | return, 退货 |
| 广告 Report | campaign, 广告 |

### 3. 智能周期检测
- `2026JanReports` → 2026年1月
- `2025Q4Reports` → 2025年Q4
- `2025Oct...` → 2025年10月

### 4. 使用方式

```bash
# 自动检测并生成报告
python run.py

# 指定目录
python amazon_report_analyzer.py --dir amazon/2026FebReports

# 指定周期
python amazon_report_analyzer.py --period "2026年2月"
```

### 5. 文件结构

### 7. 技术细节

#### 周期检测逻辑
从文件名智能提取周期，优先使用 transaction > business > returns 文件

#### 目录检测优先级
1. 当前目录本身
2. 当前目录的子目录
3. 当前目录的 amazon 子目录
4. 脚本所在目录
5. 脚本所在目录的 amazon
6. 父目录

#### 数据加载
- Business Report：包含 SKU、销量、销售额
- Transaction Report：用于补充 SKU 明细
- Returns Report：退货原因分析
- Ad Report：广告数据分析
